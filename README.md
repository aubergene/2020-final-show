## Getting started

Ensure you have Node 12.6.0 installed

```bash
npm i
```

### Running the project

```bash
npm run dev
```

## Production mode and deployment

This will export the site to `public` and then commit it to the `gh-pages` and push to Github

```bash
npm run deploy
```
